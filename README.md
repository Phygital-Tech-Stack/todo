# clawd-todo

Telegram Mini App for Clawd's daily TODO list.

**Live app:** https://phygital-tech-stack.github.io/todo

Tasks are managed by [Clawd](https://github.com/openclaw/openclaw). `tasks.json` is auto-updated whenever tasks are added or completed.

## Stack
- Pure HTML/CSS/JS — no framework, no build step
- Telegram Mini App SDK for native integration
- GitHub Pages for hosting
